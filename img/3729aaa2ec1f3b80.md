1，查看当前分支状态

```undefined
git branch
* (HEAD detached at 925fda6)
xxxx 分支
```

2，从小组分支新建一个临时 tem 分支，把当前提交的代码放到整个分支

```undefined
git branch tem
git checkout tem
```

3，换回要回到的小组分支

```undefined
git checkout 小组分支
```

4，然后 merge 刚才创建的临时分支

```css
git merge tem
Updating cad0be9..2437c6b
Fast-forward
......
```

5，检查是否有冲突，没有就提交到远端

```undefined
git push origin 小组分支
```

6，删除临时分支

```undefined
git branch -d tem
```
